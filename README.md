# Smart Expense Tracker with Insights

A Python-based expense tracking application that helps users log daily expenses, analyze spending patterns, and visualize financial insights through charts.

## Key Features

- Expense logging with date, category, amount, and description  
- Category-wise expense tracking  
- Monthly expense summaries and insights  
- Highest spending category detection  
- Recent transactions dashboard  
- Data visualization using bar charts, line graphs, and pie charts  

## Tech Stack

- Python 3.x  
- Tkinter  
- Pandas  
- Matplotlib  
- CSV File Handling  

## Program Flow

1. User enters expense details through GUI  
2. Input is validated and saved in CSV  
3. Expense data is analyzed using Pandas  
4. Dashboard displays summaries and recent transactions  
5. Charts visualize spending patterns
